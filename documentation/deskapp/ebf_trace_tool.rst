多功能调试助手【旧版C#】
------------

旧版C#野火多功能调试助手上位机开源，请看下面介绍。
为方便大家学习如何开发嵌入式中常用的上位机软件，
我们的《串口调试助手》、《GSM调试助手》、《GPS调试助手》、《网络调试助手》、《摄像头调试助手》
现在已经开放源代码，这些上位机是使用C# (.net
WPF)编写的，使用VS2013开发环境

帖子链接：http://www.firebbs.cn/forum.php?mod=viewthread&tid=11985
